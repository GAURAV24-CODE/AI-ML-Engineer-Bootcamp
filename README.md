# 🎯 About the Bootcamp

Welcome to my **AI/ML Engineer Bootcamp** repository.

This repository is a structured **45-day learning journey** where I document everything I learn—from Python fundamentals to building and deploying AI applications.

The focus is on **learning by doing**, with daily notebooks, coding practice, assignments, mini-projects, interview preparation, and real-world AI/ML projects.
